環境構築

ちょっと試してみたいだけという人は、CDNを読み込めば簡単にVueが使えます。

以下は(2018/08/16)時点での最新verです。

Vue.jsをインストールするのにnpmを使用するため、Node.jsをインストールします。

使用するOSを選んでインストーラーをダウンロードしてください

https://nodejs.org/en/download/

ダウンロードしたら実行します。

基本的に設定は全てデフォルトで問題ありません。

完了したら、コマンドプロンプトを開いてインストールが上手くいってるか確認します。

画像のように表示されていればOKです。

node --version

同時にnpmもインストールされているのでこちらも確認します。

npm --version

続いてVueをインストールしていきます。

以下のコマンドを実行してvue-cliをグローバルにインストールします。

npm install -g vue-cli

実行するとプロジェクト作成に必要なファイル等のテンプレートをインストールしてくれます。

vue --version
　vueのversionが確認できます

ここまでで、プロジェクトを立てるための準備は完了です。

それではプロジェクトを作成していきます。

以下のコマンドを任意のディレクトリに移動してから実行します。

vue init webpack <プロジェクト名>

対話形式で色々聞かれますが、全てEnterで進めて特に問題ありません。

実行完了すると実行したディレクトリに<プロジェクト名>フォルダが作成されます。

中身はこんな感じ。

<プロジェクト名>フォルダをカレントディレクトリに指定して 

npm run dev
を実行します。

以下のようになればビルドできているので、http://localhost:8080を開いてみてください。

Welcome to Your Vue.js App　のページが開けば成功です。
